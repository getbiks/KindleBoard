# Changelog

All notable changes to KindleBoard will be documented in this file.

---

## v0.1.0 - Initial Release
Released: 19 July 2026

### Added

- Flask-based dashboard server
- Raspberry Pi system statistics
- CPU usage
- CPU temperature
- RAM usage
- Disk usage
- System uptime
- Weather widget
- qBittorrent widget
- Cryptocurrency widget (BTC, ETH, XRP)
- USD and INR price display
- 12-hour clock
- Automatic dashboard image generation
- Automatic image refresh service
- TRMNL-compatible API endpoint
- KOReader TRMNL Display compatibility
- GitHub repository

### Changed

- Dashboard title changed from KindleBoard to BatCave
- Improved dashboard spacing
- Compact crypto widget layout
- Better CPU usage calculation

### Fixed

- Python virtual environment issues
- qBittorrent integration
- Font detection
- Flask service configuration
- Image rendering issues
- CPU reporting
